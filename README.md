## How to install Pydos
**Create a Folder for Pydos setup**
```bash
mkdir <folder>
```
**Download Pydos From GitHub**
```bash
git clone https://github.com/T7280H/pydos2-0
```
**Enter the installation folder**
```bash
cd pydos2-0/installation
```
**Start the Setup**
```python
python install.py
```
